DATA CENTER INVENTORY MANAGEMENT (DCIM) adalah sebuah aplikasi berbasis web yang digunakan untuk menginventarisir perangkat atau device pada Data Center.
Dilengkapi juga dengan manajemen IP Address dan SLA Data untuk ISP.

Kebutuhan Sistem
 
 * PHP Versi >= 7.3
 * MariaDB atau MySQL (versi terbaru lebih disarankan)

Username for Testing

Jangan lupa untuk mengkonfigurasi email dan database. Konfigurasi email digunakan untuk fitur reset password dan mengaktifkan user baru.

 * Konfigurasi email di '/application/config/email.php'
 * Konfigurasi database di 'application/config/database.php'
